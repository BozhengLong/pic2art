# Examples

## Directory Structure

```
examples/
├── lego/
│   ├── inspiration/    # LEGO mosaic artworks by Geoffroy Amelot
│   ├── input/         # Sample input images
│   └── output/        # Generated output images
└── (future styles will be added here)
```

## LEGO Mosaic Inspiration

The `lego/inspiration/` directory contains LEGO mosaic artworks by [Geoffroy Amelot](https://supergeoffroy.tumblr.com/archive/2013/10), which inspired this project:

- **Girl with a Pearl Earring** - Johannes Vermeer's masterpiece in LEGO
- **Mona Lisa** - Leonardo da Vinci's iconic portrait in LEGO
- **Van Gogh Self-Portrait** - Vincent van Gogh's self-portrait in LEGO

These images demonstrate the artistic potential of LEGO mosaic style.

## Usage

Place your input images in the appropriate style's `input/` directory and run the converter. Output will be saved to the `output/` directory.

将输入图片放在对应风格的 `input/` 目录，运行转换器后输出将保存到 `output/` 目录。
